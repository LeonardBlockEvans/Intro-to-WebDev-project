Setting the stage

In 1994 the World Wide Web was rapidly expanding in all directions. In this time the first bot countermeasures were established, and the first HTML validator was created. On October 1st, 1994 the W3C was founded to develop standards for the World Wide Web, standards for markup languages that would come to include HTML, XHML, XML, and CSS, and to be an educational resource. The rest of 1994 through 1995 saw the rise of many important tools and websites used today including PHP 1.0, Amazon, Internet explorer, eBay, HTML 2.0, and finally JavaScript 1.0 which was released on December 4 forth 1995.

Technology

The robots exclusion standard, also known as the robots exclusion protocol or simply robots.txt, is a standard used by websites to communicate with web crawlers and other web robots. This is one of the first bot countermeasures.

Software engineers Dan Connolly and Mark Gaither created the first version of an online HTML validator. The tool served to check the validity of documents, i.e. compare the document code with valid standards and look for potential errors.

The World Wide Web Consortium (W3C) is the main international standards organization for the World Wide Web. Founded in 1994 and currently led by Tim Berners-Lee, the consortium is made up of member organizations that maintain full-time staff working together in the development of standards for the World Wide Web.

The very first incarnation of PHP was a simple set of Common Gateway Interface (CGI) binaries written in the C programming language.

Internet Explorer is Microsoft's venture into the Web Browser market. The initial version of Internet Explorer was incarnated from Spyglass Mosaic.

In April 1994, Dan Connolly created a new draft for the language, then named 'HTML 2.0'.

Brendan Eich of Netscape designed the first version of JavaScript in 1995. JavaScript later become the basis for other programming languages, such as ActionScript used in Macromedia Flash. In 1998, JavaScript was standardized by ISO.

Websites

Jeff Bezos, the founder of Amazon, started the venture as an online book retailer on July 5, 1994, but the business has expanded over the years to sell almost anything one can think of and is now one of the most valuable companies in the world.

eBay was founded in Pierre Omidyar's San Jose living room back in September 1995. It was from the start meant to be a marketplace for the sale of goods and services for individuals.
