Foundation

The idea of a World Wide Web originated in 1989 when Tim Berners-Lee, a researcher for CERN, proposed the creation of a global hypertext project, which by 1993 evolved into what we know as the World Wide Web. Major events between 1989 and 1993 include the WWW Virtual Library, the publishing of a document entitled HTML Tags, HTTP v0.9, ViolaWWW, BBEdit HTML and text editor, and the popularization of search engines.

People

Sir Timothy John Berners-Lee also known as TimBL, is an English computer scientist best known as the inventor of the World Wide Web.

The first website at CERN – and in the world – was dedicated to the World Wide Web project itself and was hosted on Berners-Lee's NeXT computer.

Technology

The internet stems from the proposal for a Global hypertext project.

A Little History of the World Wide Web.

The WWW Virtual Library is the oldest catalogue of the Web started by Tim Berners-Lee, the creator of HTML and of the Web itself.

HTML1 is a name for Tim Berners-Lee's original HTML. These tags are taken from a document titled "HTML Tags" , no author shown but we assume it was written by Tim Berners-Lee.

The Hypertext Transfer Protocol (HTTP) is an Internet protocol designed to exchange hypertext documents (images, websites, etc.) between the server and the web browser. The first protocol design, later called HTTP v0.9, was created by Tim Berners-Lee as part of the WWW service specification. The HTTP v0.9 version was extremely simple, using only the GET method with one parameter, i.e. the name of the requested document.

Pei-Yuan Wei developed the ViolaWWW browser for Unix while he was working at the University of California at Berkeley.

On April 12 1992, Veteran Mac developer Rich Siegel announces the release of “BBEdit”, a new freeware text editor for the Mac.
